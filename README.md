<p dir='rtl' align='right'>برای این تابع از دیتاست شماره ۱ استفاده کردیم. این دیتاست تمامی پست‌های صفحه خبری ABCnews که از تاریخ 2012 تا 2016 گذاشته شده است را نشان می‌دهد. این پست‌ها ۴ نوع مختلف دارند:</p>

•	Status <br />

•	Video <br/>
